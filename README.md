# Marcapáginas
#### Video Demo: https://youtu.be/pZO5MxTC5ZE
#### Description: Marcapáginas es un gestor de lectura para ayudar a los usuarios a organizar y mantener un registro de sus libros favoritos. Con una interfaz fácil de usar, Marcapáginas permite a los usuarios añadir, editar y eliminar marcadores. El objetivo de este proyecto es ofrecer una solución sencilla pero eficaz para la gestión de marcapáginas, facilitando el acceso y el uso compartido de contenidos en línea. Index.hmlt es la página de inicio de la aplicación. La página login.html es la página de inicio de sesión de la aplicación, esta permite a los usuarios existentes acceder a su cuenta. Por otro lado, la página register.html es la página de registro en la que los usuarios nuevos pueden crear una cuenta y empezar a utilizar la aplicación. La página main.html no tiene funcionalidad adicional más allá de presentar una interfaz de usuario básica y proporcionar enlaces a otras páginas de la aplicación. La página search.html es una página de búsqueda de libros que permite a los usuarios buscar libros y mostrar los resultados de la búsqueda, con la opción de agregar los libros a su biblioteca en biblioteca.html por medio de una conexión desde la Google Books API hacia la base de datos Marcapag.db, y echar un vistazo previo al contenido del libro. La aplicación hace uso de la API de Google Books, conexión que es realizada en varios archivos: book.html y search.html. En book.html, la API de Google Books se carga  mediante el script <script type="text/javascript" src="https://www.google.com/books/jsapi.js"></script>. Luego, se utiliza la función google.books.load() para cargar la API y la función google.books.DefaultViewer para crear un visor de libros que se carga con la información del libro seleccionado. En search.html, se usa la API de Google Books para buscar libros mediante la función fetch que hace una solicitud GET a la API de Google Books con la consulta de búsqueda proporcionada por el usuario. Luego, se procesa la respuesta JSON y se muestra la lista de resultados en la página.
